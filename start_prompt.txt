Создай простой консольный чат на Kotlin для взаимодействия с AI через API.

Требования:
1. Приложение должно быть консольным (CLI)
2. Пользователь вводит промпты через консоль
3. Ответы от AI выводятся в консоль
4. Использовать API Cursor (или уточнить, какой именно AI API нужно использовать - OpenAI, Anthropic, и т.д.)
5. Код должен быть на Kotlin
6. Приложение должно работать в интерактивном режиме - пользователь может отправлять несколько сообщений подряд
7. Должна быть возможность выхода из программы (например, команда "exit" или "quit")
8. Обработка ошибок при запросах к API
9. Показывать индикатор загрузки или сообщение о том, что идет обработка запроса

Структура проекта:
- Основной файл с точкой входа
- Класс для работы с API
- Класс для обработки пользовательского ввода
- Конфигурация для API ключа (можно через переменные окружения или файл конфигурации)

Дополнительно:
- Добавь README с инструкциями по запуску на MacOS
- Используй современные практики Kotlin (coroutines для асинхронных запросов, если нужно)
- Код должен быть чистым и понятным
